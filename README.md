# Gardener

A Clojurescript sdk.

## Getting Started

The following tasks (lein aliases) are available for contributors to develop and
test Clj/Cljs locally:

Build & Watch Cljs

	lein build-cljs

Start a node repl

	lein node-repl

Run Clj tests, along with a test runner

	lein test-clj

Run Cljs tests (on Node)

	lein test-cljs
